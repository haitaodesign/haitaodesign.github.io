什么是Markdown?

Markdown 的目标是实现「易读易写」,能让文档更容易读、写和随意改。
Markdown 的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。
Markdown 语法的目标是：成为一种适用于网络的书写语言。
